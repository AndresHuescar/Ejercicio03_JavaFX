# Ejercicio 3 - JavaFX

## Descripción

Este proyecto corresponde al ejercicio 3 del bloque BC5 de JavaFX.

El objetivo del ejercicio es crear una ventana que muestre un mensaje diferente al hacer clic en un botón.

## Tecnologías utilizadas

- Java
- JavaFX
- Maven
- IntelliJ IDEA
